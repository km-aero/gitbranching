# Git branching

making branches, push branches, merge branches

# create new branches

```
git checkout -b <name>
```

naming convention
```
dev-description-of-feature
```

# Flow
1. make branch
2. code, git add, git commit
3. push to branch

## git commands
- git checkout <branch> (moves to branch)
- git branch (lists branches)
